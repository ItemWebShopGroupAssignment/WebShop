Välkommen till Game Web Shop

Det största varuhuset för spel

Här kan du söka efter intressanta spel som du kan köpa eller ladda ner.
Om du inte redan är kund hos GWS kan du öppna ett konto genom att klicka här.
För att handla gör du så här:

Välj produkterna du vill handla genom att klicka på Köp-knappen.
Klicka på "Till kassan" vid varukorgen uppe till höger.
Fyll i dina inloggningsuppgifter.
Upplever du problem att genomföra ditt köp får vi be att du kontaktar oss.

Genom att logga in på ditt kundkonto och gå in under fliken "Beställningar" får du snabbt en överblick över hur långt i behandlingen din order kommit.
Är det så att din order inte skickats ännu eller att du inte mottagit den trots att den skickats får vi be att du kontaktar oss så hjälper vi dig vidare.
